### 送万字文档+配件销售业绩管理系统+SSM+JSP
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、JSP、JavaScript、CSS、文件上传
总共11张数据库表

系统功能模块：
本系统包含管理员和员工2个角色。
管理员角色：
1. 客户管理：添加、删除、查询和修改客户信息。
2. 供应商管理：添加、删除、查询和修改供应商信息。
3. 配件管理：添加、删除、查询和修改配件信息。
4. 出售信息管理：查看配件的销售信息，并生成销售报表。
5. 进货信息管理：查看配件的进货信息。
员工角色：
1. 进货信息管理：登记配件进货信息，查询配件的进货信息。
2. 配件管理：添加、查询和修改配件信息，但不能删除配件。
3. 出售信息管理：登记配件出售信息，查询员工本人已出售的配件信息。
